Politique de confidentialité

Dernière mise à jour : 7 mai 2026

La présente politique de confidentialité décrit comment l'application Run Expert collecte, utilise et protège vos informations personnelles.

1. Responsable du traitement

L'Application Run Expert est développée et exploitée par E-3P. Pour toute question : contact@runexpert.fr

2. Données collectées

  • Données de compte : prénom, adresse e-mail.
  • Données d'activité : distance, durée, allure, GPS.
  • Données de santé estimées : VO2max, zones cardiaques.
  • Données de localisation : GPS pendant une session active uniquement.

3. Finalités

  • Suivi GPS et plans d'entraînement.
  • Amélioration de l'expérience utilisateur.
  • Fonctionnalités communautaires (avec consentement).

4. Partage des données

Nous ne vendons pas vos données. Sous-traitants : Firebase/Google (authentification, stockage), Expo/EAS (distribution).

5. Localisation

Le GPS est utilisé uniquement pendant une session de course. Vous pouvez révoquer cette autorisation dans les paramètres de votre appareil.

6. Sécurité

Données transmises via HTTPS/TLS. Mesures techniques appropriées en place.

7. Conservation

Données conservées tant que votre compte est actif. Suppression sur demande à contact@runexpert.fr.

8. Vos droits (RGPD)

  • Accès, rectification, effacement, portabilité, opposition.

Pour exercer ces droits : contact@runexpert.fr

9. Publicités

L'Application affiche des publicités via Google AdMob. Vous pouvez désactiver la personnalisation dans Paramètres → Google → Publicités.
